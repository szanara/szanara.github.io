# Site pessoal com GitHub Pages

Arquivos principais:

- `index.html`: estrutura do site
- `styles.css`: visual do site

## Como personalizar

Troque no `index.html`:

- `Seu Nome`
- `Sua profissão ou área`
- texto da apresentação
- links de GitHub, LinkedIn e e-mail
- foto de perfil
- projetos, experiência e contato

## Como publicar

1. Crie um repositório no GitHub com o nome `seuusuario.github.io`
2. Envie estes arquivos para esse repositório
3. Aguarde alguns minutos
4. Seu site vai abrir em `https://seuusuario.github.io/`
